================================================================================
TROUBLESHOOTING
================================================================================

If Projects/Tasks page is still black:
1. Check browser console (F12) for errors
2. Check backend terminal for errors
3. Verify backend is running on port 3001
4. Verify frontend is running on correct port
5. Try hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac)

If data doesn't appear after creation:
1. Check browser console for network errors
2. Check backend terminal for API errors
3. Verify database is running
4. Try refreshing the page

If table is cut off or not visible:
1. Check browser zoom level (should be 100%)
2. Try resizing browser window
3. Check for console errors about layout

If horizontal scrolling doesn't work:
1. Verify table has min-w-[1400px] class
2. Check that overflow-x-auto is applied
3. Try resizing browser window narrower
