## How It Works

### Automatic Analysis Pipeline
```
File Save/Edit
    ↓
Pattern Matching (Hook triggers)
    ↓
Diagnostic Check (Built-in tools)
    ↓
AI Analysis (Claude Haiku)
    ↓
Report Generation
    ↓
Suggestions & Fixes
    ↓
User Review & Apply
```

### Manual Analysis Pipeline
```
User Request
    ↓
Context Gathering (Codebase exploration)
    ↓
AI Analysis (Multi-file context)
    ↓
Detailed Report
    ↓
Actionable Recommendations
    ↓
Implementation Guidance
```
