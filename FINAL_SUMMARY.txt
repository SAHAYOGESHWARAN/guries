SUPPORT & TROUBLESHOOTING
================================================================================

Issue: 405 Method Not Allowed
Solution: Ensure /api directory is properly structured with auth.ts, health.ts, and v1/[[...route]].ts

Issue: API Returns HTML Instead of JSON
Solution: Check that API routes are in /api directory, not root

Issue: Frontend Shows Blank Screen
Solution: Check browser console for errors, verify VITE_API_URL is set to /api/v1

Issue: Login Fails with 401
Solution: Verify credentials (admin@example.com / admin123), check JWT_SECRET is set

Issue: CORS Errors
Solution: Verify CORS headers in API handlers are properly set

For more help, see DEPLOYMENT_GUIDE.md
