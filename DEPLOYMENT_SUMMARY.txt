================================================================================
                        ENVIRONMENT VARIABLES
================================================================================

REQUIRED:
  DB_HOST              - Database host (e.g., db.supabase.co)
  DB_PORT              - Database port (default: 5432)
  DB_USER              - Database user (default: postgres)
  DB_PASSWORD          - Database password (STRONG!)
  DB_NAME              - Database name (default: postgres)
  NODE_ENV             - Set to: production
  FRONTEND_URL         - Your domain (e.g., https://myapp.com)
  CORS_ORIGIN          - Your domain (e.g., https://myapp.com)
  JWT_SECRET           - Generated secret (32+ characters)
  JWT_EXPIRY           - Token expiry (e.g., 7d)
  VITE_API_URL         - Set to: /api/v1

OPTIONAL:
  VITE_GEMINI_API_KEY  - For chatbot feature
  SMTP_HOST            - For email notifications
  SMTP_PORT            - For email notifications
  SMTP_USER            - For email notifications
  SMTP_PASS            - For email notifications
