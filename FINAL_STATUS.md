## Deployment Instructions

### Quick Start (5 minutes)

1. **Initialize Database**
```bash
cd backend
node init-production-db.js
```

2. **Install Dependencies**
```bash
npm install
cd ../frontend
npm install
```

3. **Build Frontend**
```bash
npm run build
```

4. **Start Backend**
```bash
cd ../backend
npm start
```

5. **Serve Frontend**
```bash
cd ../frontend
npm run preview
```

### Verify Deployment
```bash
cd backend
node verify-qc-fix.js
```

## Support & Troubleshooting

### Common Issues

**QC Review Failing**
- Check `.env.production` has correct `VITE_API_URL`
- Verify backend is running
- Check browser console for errors

**Database Issues**
- Run `node init-production-db.js`
- Check database file exists
- Verify schema with `node verify-qc-fix.js`

**API Not Responding**
- Verify backend is running on correct port
- Check CORS configuration
- Review backend logs
