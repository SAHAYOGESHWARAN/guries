================================================================================
AUTHENTICATION & AUTHORIZATION
================================================================================

AUTHENTICATION METHOD: JWT (JSON Web Tokens)
- Token Expiration: 7 days
- Token Storage: localStorage
- Token Format: Bearer {token}
- Refresh: Not implemented (7-day expiration)

ROLES & PERMISSIONS:
1. Admin
   - Full system access
   - QC review and approval
   - User management
   - Audit log access

2. QC
   - QC review and approval
   - View all assets
   - Cannot manage users

3. Manager
   - Team management
   - View team assets
   - Submit for QC
   - View reports

4. User
   - Create/edit own assets
   - Submit for QC
   - View own assets

5. Guest
   - View-only access
   - Cannot create/edit

MIDDLEWARE:
- verifyToken - Validates JWT
- requireAdmin - Checks admin role
- requireRole(role) - Checks specific role
- requirePermission(permission) - Checks permission
- requireQCPermission - Checks QC permission
