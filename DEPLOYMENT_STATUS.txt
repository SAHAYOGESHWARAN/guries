================================================================================
TROUBLESHOOTING
================================================================================

Frontend Won't Load:
  1. Check port 5173 is available
  2. Verify npm run dev:frontend is running
  3. Clear browser cache
  4. Check console for errors

Backend Won't Start:
  1. Check port 3003 is available
  2. Verify npm run dev:backend is running
  3. Check database file exists
  4. Review backend logs

Database Errors:
  1. Verify backend/mcc_db.sqlite exists
  2. Check file permissions
  3. Delete and restart to reinitialize
  4. Check backend logs for errors

API Errors:
  1. Verify backend is running
  2. Check CORS configuration
  3. Verify authentication token
  4. Check API endpoint URL
