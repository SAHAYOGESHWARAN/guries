================================================================================
TROUBLESHOOTING GUIDE
================================================================================

If deployment fails:
  1. Check Vercel build logs for errors
  2. Verify environment variables are set
  3. Check for missing dependencies
  4. Verify database connection string
  5. Check for TypeScript compilation errors

If frontend shows 404:
  1. Verify vercel.json routes configuration
  2. Check SPA fallback route
  3. Verify frontend/dist/index.html exists
  4. Check browser cache (Ctrl+Shift+Delete)

If API endpoints fail:
  1. Verify api/index.ts exports correctly
  2. Check backend routes configuration
  3. Verify environment variables
  4. Check database connection
  5. Review backend logs

If Socket.io fails:
  1. Check CORS configuration
  2. Verify VITE_SOCKET_URL environment variable
  3. Check browser console for errors
  4. Verify Socket.io server is running
  5. Check firewall/proxy settings
