                        HOW DATA FLOWS IN PRODUCTION
================================================================================

1. User Request:
   Frontend makes request to /api/v1/projects

2. Vercel Routing:
   Vercel routes /api/* to api/backend-proxy.ts

3. Backend Proxy:
   api/backend-proxy.ts receives request
   Sets CORS headers
   Calls Express app

4. Express Backend:
   Routes to appropriate controller
   Queries database
   Returns JSON response

5. Frontend Display:
   Receives data from API
   Renders in TasksView or ProjectsView
   Displays in table with proper formatting
